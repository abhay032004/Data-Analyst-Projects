# Data Analyst Projects

## Overview
This folder contains projects focused on **Data Analysis and Business Intelligence**. The projects demonstrate the ability to collect, clean, analyze, and visualize data to extract actionable business insights.

The emphasis is on understanding data patterns, trends, and metrics that support decision-making using analytical tools and visualization techniques.

## Key Skills Demonstrated
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization and reporting
- Business insights generation
- SQL-based data querying

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL / MySQL
- Power BI
- Excel
- Matplotlib, Seaborn

## Ideal For Roles
- Data Analyst
- Business Analyst
- Reporting Analyst
