# Project Title
Diwali Sales Analysis Using Python

## Project Description
This project focuses on analyzing Diwali sales data using Python to understand customer purchasing behavior. Through data cleaning, preprocessing, and exploratory data analysis (EDA), the project identifies key trends across demographics, locations, occupations, and product categories to support data-driven business decisions.

## Dataset Overview
The dataset contains customer-level Diwali sales information, including:
- Gender, Age Group, Marital Status
- State and Occupation
- Product Category and Product ID
- Number of Orders and Purchase Amount

The data is structured and suitable for detailed exploratory analysis.

## Project Vision
The vision of this project is to demonstrate how data analytics can uncover actionable insights from sales data, helping businesses optimize marketing strategies, target the right customer segments, and improve overall sales performance during festive seasons.

## Technology Stack
- Programming Language: Python  
- Libraries: Pandas, NumPy  
- Visualization: Matplotlib, Seaborn  

## Approach & Methodology
- Data loading and inspection
- Data cleaning (handling null values and removing irrelevant columns)
- Data type conversion and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of sales trends by:
  - Gender
  - Age Group
  - State
  - Marital Status
  - Occupation
  - Product Category

## Key Features
- End-to-end Exploratory Data Analysis (EDA)
- Insightful visualizations using Seaborn and Matplotlib
- Customer segmentation analysis
- Identification of top-performing states and product categories
- Clear, business-focused insights
